# SLM Mail Campaign Analysis

## Overview
This project analyzes the performance of a direct mail marketing campaign using transaction, mail, and holdout datasets.

The analysis includes:
- Data cleaning and validation
- Address matching between mailed and holdout groups
- Response rate calculation
- Cost per order (CPO)
- Return on ad spend (ROAS)
- Test cell comparison
- Strategic recommendations for future campaigns

## Files
- `analysis.ipynb` – Main analysis notebook
- `EDA.ipynb` – Exploratory data analysis
- Raw data files are excluded due to confidentiality

## Tools Used
- Python (pandas, matplotlib, numpy, plotly)
- Jupyter Notebook

## Notes
This repository demonstrates methodology and analytical reasoning.  
Raw campaign data is not included.
